
# DushBot

A simple WhatsApp bot using whatsapp-web.js

## Features

- Say `hi` to get a greeting

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start the bot:
   ```
   node index.js
   ```

3. Copy the QR code link printed in the terminal and open it in a browser tab. Scan it with WhatsApp.
